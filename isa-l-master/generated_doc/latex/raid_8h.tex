\subsection{Detailed Description}
Interface to R\-A\-I\-D functions -\/ X\-O\-R and P+\-Q calculation. This file defines the interface to optimized X\-O\-R calculation (R\-A\-I\-D5) or P+\-Q dual parity (R\-A\-I\-D6). Operations are carried out on an array of pointers to sources and output arrays.
